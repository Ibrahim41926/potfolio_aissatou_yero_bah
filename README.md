# ePortfolio — Aissatou Yero Bah

Petit projet que j'ai réalisé pour une amie, **Aissatou Yero Bah**, étudiante en L1 AES à l'Université Rennes 2.

Elle avait besoin d'un ePortfolio dans le cadre d'un devoir universitaire, alors je lui ai créé un site web simple, propre et responsive, hébergé sur GitHub Pages.

## Ce que contient le site

- **Présentation** — qui elle est, ses infos de base
- **Parcours** — timeline de son parcours scolaire de la Guinée jusqu'à Rennes
- **Situation d'apprentissage** — grille de réflexivité sur son stage de terrain à l'Université de Boké

## Stack

HTML + CSS pur, aucune dépendance. Une seule page, prête pour GitHub Pages.
